# 🧠 AI Resume Intelligence Platform

### Futuristic Resume Analysis System for Modern Hiring

A **next-generation resume intelligence platform** that analyzes resumes
against job descriptions and provides **ATS scoring, skill insights, and
improvement recommendations** using an advanced NLP-inspired analysis
engine.

Built with **HTML, CSS, and JavaScript**, the platform simulates a
**modern AI recruitment system** with interactive dashboards, 3D
visualization, and real-time analysis feedback.

------------------------------------------------------------------------

# 🚀 Project Vision

Hiring today is driven by **Applicant Tracking Systems (ATS)** and
automated screening tools.\
Most candidates do not know **why their resume gets rejected**.

This project bridges that gap by building a **smart resume intelligence
dashboard** that:

-   Evaluates resumes against job descriptions
-   Detects missing skills and keywords
-   Provides ATS-style scoring
-   Visualizes resume strength through graphs and analytics

------------------------------------------------------------------------

# ✨ Key Features

## 📄 Resume Upload & Parsing

-   Upload resume in **PDF format**
-   Automatic **text extraction using PDF.js**
-   Drag-and-drop upload
-   Resume preview before analysis

------------------------------------------------------------------------

## 🧠 Smart Resume Analysis Engine

The platform includes a **custom NLP-inspired analysis engine** capable
of:

-   Skill extraction
-   Technology recognition
-   Keyword matching
-   Resume structure evaluation
-   ATS score calculation

The system detects **180+ technologies** across categories:

-   Programming Languages
-   Web Development
-   Machine Learning
-   Databases
-   Cloud & DevOps
-   Developer Tools
-   Soft Skills

------------------------------------------------------------------------

## 📊 ATS Compatibility Score

The resume is evaluated using weighted scoring:

  Metric                 Weight
  ---------------------- --------
  Skills Match           40%
  Keyword Match          25%
  Experience Relevance   20%
  Content Quality        10%
  Resume Completeness    5%

The final score estimates **how likely a resume is to pass automated ATS
screening systems**.

------------------------------------------------------------------------

## 🔍 Job Description Matching

The analyzer compares the resume with a job description to detect:

-   Missing technologies
-   Keyword gaps
-   Skill alignment
-   Experience relevance

# ⚙️ How It Works

### Step 1

Upload a **PDF resume**.

### Step 2

Paste the **target job description**.

### Step 3

The system performs analysis:

Resume Parsing → Skill Extraction → Keyword Matching → Resume Quality
Evaluation → ATS Score Calculation

### Step 4

Results appear on the **interactive dashboard**.
